# Глосарій термінів (Згенеровано та верифіковано ШІ)

Тут зібрані ключові поняття, які допоможуть краще зрозуміти логіку систем числення.

| Термін | Визначення |
| :--- | :--- |
| **Система числення** | Сукупність правил запису чисел за допомогою обмеженого набору символів (цифр). |
| **Основа системи** | Кількість різних знаків (цифр), що використовуються для запису числа в позиційній системі. |
| **Алфавіт системи** | Повний набір символів (цифр, літер), які використовуються для представлення чисел. |
| **Позиційна система** | Система, у якій значення кожної цифри залежить від її місця (позиції) у числі. |
| **Двійкова система** | Позиційна система числення з основою 2 (використовуються цифри 0 та 1). |
| **Біт (Bit)** | Найменша одиниця інформації в двійковій системі. |
