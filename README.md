# Навчальний модуль: Системи числення та переведення
**Предмет:** Інформатика, 10 клас (Базовий рівень)
**Автор:** Ващенко Дар'я Сергіївна

## 🎯 Мета проєкту
Створення інтерактивного навчального контенту для опанування учнями методів переведення чисел між позиційними системами числення.

## 🤖 Використання ШІ
У цьому проєкті було застосовано наступні інструменти штучного інтелекту:
- **Gemini (Google):** Розробка структури курсу, генерація теоретичних матеріалів, створення тестів та практичних завдань.
- **Canva Magic Media / DALL-E:** Генерація візуальних схем та ілюстрацій.
- **Claude:** Оптимізація коду для Markdown та створення структури репозиторію.

## 📂 Структура репозиторію
- `/theory` — теоретичні відомості та глосарій.
- `/practice` — лабораторні роботи та практичні завдання.
- `/tests` — тести для самоперевірки.
- `/resources` — посилання на додаткові матеріали та зображення.
